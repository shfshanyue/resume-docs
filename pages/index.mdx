# 文档

欢迎使用一纸简历！这是一个强大的简历制作工具，结合了 AI 技术和 Markdown 语法的强大功能，帮助您轻松创建专业、吸引人的简历。

## 主要特点

1. **AI 驱动**：利用先进的人工智能技术，提供智能生成、优化和评估功能。
2. **Markdown 支持**：使用简洁的 Markdown 语法快速编写和格式化您的简历。
3. **Tailwind CSS 集成**：提供丰富的样式选项，轻松自定义简历外观。
4. **Lucide 图标支持**：添加专业图标，增强简历的视觉吸引力。

## 核心功能

- [使用语法](/usage)：学习如何使用 Markdown 和特殊语法创建简历。
- [智能生成](/ai-generate)：利用 AI 快速生成符合行业标准的简历内容。
- [一键优化](/optimize)：AI 驱动的简历优化，提升您的职业竞争力。
- [术语检查](/case-error)：确保技术术语拼写正确，展现专业形象。
- [自动一页](/autofit)：智能调整简历长度，确保内容恰好填满一页或两页。
- [简历评估](/evaluate)：获取 AI 提供的详细评估和面试问题建议。

## 开始使用

1. 访问[一纸简历](https://cv.devtool.tech)网站。
2. 选择新建简历或使用 AI 智能生成功能。
3. 使用 Markdown 语法编写您的简历内容。
4. 利用 Tailwind CSS 类和 Lucide 图标美化您的简历。
5. 使用一键优化和评估功能完善您的简历。

无论您是应届毕业生、经验丰富的专业人士，还是正在寻求职业转型的求职者，一纸简历都能满足您的需求。开始使用吧，让 AI 和 Markdown 的力量助您在竞争激烈的就业市场中脱颖而出！
